Override the default 404 page with any page from the Admin Panel.

== Description ==

Allows users to replace the default 404 page with a custom page from the Pages section in the Admin Panel.

Just select the desired page from the drop-down list and Save! And that's it!

== Installation ==

1. Extract the downloaded ZIP file.
2. Copy the custom-404-pro folder to the wp-content/plugins directory.
3. Activate from the Plugins Section.
4. Options are available under: Settings > Custom 404 Pro

== Screenshots ==

1. Activate the plugin from the WordPress Admin Panel
2. Go to Settings > Custom 404 Pro to open up the Options Page
